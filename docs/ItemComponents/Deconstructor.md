# Deconstructor


## 属性

| 属性|类型|默认值|描述 |
| ---|---|---|--- |
| DeconstructItemsSimultaneously|bool|false| |
| DeconstructionSpeed|float|1.0| |

此组件还支持以下位置定义的属性: [Powered](Powered.md), [ItemComponent](ItemComponent.md)
